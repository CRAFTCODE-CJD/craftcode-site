Atlas Editor Workflow

// Last updated  2026-04-20
// Engine        UE 5.5
// Plugin v      2.1
// Reading time  ~ 4 min

The Atlas Editor lets you pack multiple sprites into a single texture atlas with configurable padding, power-of-two alignment, and automatic slice generation. This guide walks through a typical import-to-export workflow.

If you're new to the plugin, start with the Getting Started guide — it covers installation and project-wide settings that apply to every atlas you'll build.

Step 1: Declare your sprite array

Define a UPROPERTY array that the editor can populate with the sprites you want to pack:

// MyAtlasBuilder.h
#pragma once

#include "CoreMinimal.h"
#include "SpriteOptimizer/AtlasTypes.h"

class SPRITEOPT_API AMyAtlasBuilder : public AActor
{
    GENERATED_BODY()

public:
    UPROPERTY(EditAnywhere, Category = "Atlas")
    TArray<UTexture2D*> Sprites;

    UPROPERTY(EditAnywhere, Category = "Atlas", meta = (ClampMin = "2"))
    int32 Padding = 4;
};

The ClampMin meta prevents negative padding — see Settings Reference for the full list of meta specifiers the plugin honours.

// Note
Atlas size is always rounded up to the nearest power of two. For a strict size, use Fixed Atlas mode instead — it trades packing efficiency for predictability.

Step 2: Run the optimizer

Right-click your actor in the Outliner and select Sprite Optimizer → Build Atlas. The editor opens a preview window showing packed layout, wasted pixels, and estimated VRAM savings. Use the Cross-Engine Export dialog to save the atlas as Unity TexturePacker JSON, Godot `.tres`, or plain PNG + `.json`.

Step 3: Verify the generated slices

The plugin writes a USpriteAtlasSlice asset per sprite, with UV bounds pre-computed. Reference these from your materials:

// MyAtlasBuilder.cpp
void AMyAtlasBuilder::ApplyAtlasToMaterial(UMaterialInstanceDynamic* MID, int32 SliceIndex)
{
    if (!MID || !Atlas) return;

    const FAtlasSlice& Slice = Atlas->GetSlice(SliceIndex);
    MID->SetTextureParameterValue("AtlasTexture", Atlas->Texture);
    MID->SetVectorParameterValue("UVBounds", Slice.UVBounds);
}
