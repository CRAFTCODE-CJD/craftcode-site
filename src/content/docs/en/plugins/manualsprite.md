**Manual Sprite Editor Tools** is a comprehensive plugin for Unreal Engine that
provides a dedicated editor for creating and managing custom 2D geometry for
`PaperSprite` assets. Indispensable for developers who need precise control
over their sprite meshes — especially for generating optimised `StaticMesh` and
`SkeletalMesh` assets from 2D art.

> Full Russian docs: see [the Russian version](/plugins/manualsprite/).

## Key features

- **Dedicated asset editor** — professional standalone editor for Manual
  Sprite assets with viewport, details panel, feature-rich toolbar.
- **Full geometry control** — manually add/delete/move vertices. Create
  triangles from any three selected vertices.
- **Undo/redo + copy/paste** — for all geometry operations.
- **Mirror editing** on X, Y, or both axes for symmetrical work.
- **Grid + snapping** for precise vertex placement.
- **Mesh generation** — generate StaticMesh or SkeletalMesh with a single
  click (Ctrl+M). Full control over pivot point. Automatic material creation
  (Lit/Unlit, Two-Sided).
- **Delaunay triangulation** — automatic for selected vertices (hotkey `3`).
  Intersection validation highlights overlapping edges in red.
