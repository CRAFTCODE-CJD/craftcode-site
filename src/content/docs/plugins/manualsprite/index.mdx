## <Trans id="plugin.manualsprite.h.зачем-это-нужно">Зачем это нужно</Trans>

<Block type="info">
<Trans id="plugin.manualsprite.block.p1">Стандартный Paper2D в Unreal хорош для простых спрайтов, но не даёт точного контроля над геометрией. ManualSprite добавляет новый ассет `Manual Sprite` с собственным редактором:</Trans>

- <Trans id="plugin.manualsprite.block.p2">**Pixel-perfect коллизии** — ставь вершины там, где они реально нужны.</Trans>
- <Trans id="plugin.manualsprite.block.p3">**Оптимизированная сложность** — никаких лишних треугольников.</Trans>
- <Trans id="plugin.manualsprite.block.p4">**Генерация 3D-мешей** (`StaticMesh` / `SkeletalMesh`) прямо из 2D — с материалами и базовым скелетом.</Trans>
</Block>

## <Trans id="plugin.manualsprite.h.ключевые-возможности">Ключевые возможности</Trans>

<CardGrid>
  <Card title="Выделенный редактор ассетов">
    Отдельное окно с вьюпортом, Details-панелью и многофункциональным тулбаром — не встраивается в Paper2D editor.
  </Card>
  <Card title="Полный контроль геометрии">
    Ручное добавление/удаление/перемещение вершин. Треугольники строятся из любых трёх выделенных точек.
  </Card>
  <Card title="Генерация мешей">
    `StaticMesh` или `SkeletalMesh` за один `Ctrl+M` — с контролем пивота (Center / Bottom / Origin / Custom) и авто-материалами (Lit / Unlit, двусторонние).
  </Card>
  <Card title="Триангуляция Делоне">
    Клавиша `3` — авто-триангуляция выделенных вершин. Клавиша `V` — проверка пересечений (подсвечиваются красным).
  </Card>
  <Card title="Зеркальное редактирование">
    Оси X, Y или обе одновременно — ускоряет симметричные персонажи и объекты вдвое.
  </Card>
  <Card title="Сетка и снапинг">
    Видимая сетка + настраиваемый шаг привязки. `G` — тоггл сетки, `Ctrl+G` — тоггл снапа.
  </Card>
  <Card title="Undo / Redo">
    Полная история всех операций с геометрией — безопасно экспериментируй.
  </Card>
  <Card title="Content Browser integration">
    Создавай Manual Sprite ассеты прямо из меню правой кнопки → Paper2D.
  </Card>
</CardGrid>
