## <Trans id="plugin.manualsprite.h.зачем-это-нужно">Зачем это нужно</Trans>

<Block type="info">
<Trans id="plugin.manualsprite.block.p1">Стандартный Paper2D в Unreal хорош для простых спрайтов, но не даёт точного контроля над геометрией. ManualSprite добавляет новый ассет `Manual Sprite` с собственным редактором:</Trans>

- <Trans id="plugin.manualsprite.block.p2">**Pixel-perfect коллизии** — ставь вершины там, где они реально нужны.</Trans>
- <Trans id="plugin.manualsprite.block.p3">**Оптимизированная сложность** — никаких лишних треугольников.</Trans>
- <Trans id="plugin.manualsprite.block.p4">**Генерация 3D-мешей** (`StaticMesh` / `SkeletalMesh`) прямо из 2D — с материалами и базовым скелетом.</Trans>
</Block>

## <Trans id="plugin.manualsprite.h.ключевые-возможности">Ключевые возможности</Trans>

<CardGrid>
  <Card title="Выделенный редактор ассетов" titleId="plugin.manualsprite.card.1.title">
    <Trans id="plugin.manualsprite.card.1.body">Отдельное окно с вьюпортом, Details-панелью и многофункциональным тулбаром — не встраивается в Paper2D editor.</Trans>
  </Card>
  <Card title="Полный контроль геометрии" titleId="plugin.manualsprite.card.2.title">
    <Trans id="plugin.manualsprite.card.2.body">Ручное добавление/удаление/перемещение вершин. Треугольники строятся из любых трёх выделенных точек.</Trans>
  </Card>
  <Card title="Генерация мешей" titleId="plugin.manualsprite.card.3.title">
    <Trans id="plugin.manualsprite.card.3.body">`StaticMesh` или `SkeletalMesh` за один `Ctrl+M` — с контролем пивота (Center / Bottom / Origin / Custom) и авто-материалами (Lit / Unlit, двусторонние).</Trans>
  </Card>
  <Card title="Триангуляция Делоне" titleId="plugin.manualsprite.card.4.title">
    <Trans id="plugin.manualsprite.card.4.body">Клавиша `3` — авто-триангуляция выделенных вершин. Клавиша `V` — проверка пересечений (подсвечиваются красным).</Trans>
  </Card>
  <Card title="Зеркальное редактирование" titleId="plugin.manualsprite.card.5.title">
    <Trans id="plugin.manualsprite.card.5.body">Оси X, Y или обе одновременно — ускоряет симметричные персонажи и объекты вдвое.</Trans>
  </Card>
  <Card title="Сетка и снапинг" titleId="plugin.manualsprite.card.6.title">
    <Trans id="plugin.manualsprite.card.6.body">Видимая сетка + настраиваемый шаг привязки. `G` — тоггл сетки, `Ctrl+G` — тоггл снапа.</Trans>
  </Card>
  <Card title="Undo / Redo" titleId="plugin.manualsprite.card.7.title">
    <Trans id="plugin.manualsprite.card.7.body">Полная история всех операций с геометрией — безопасно экспериментируй.</Trans>
  </Card>
  <Card title="Content Browser integration" titleId="plugin.manualsprite.card.8.title">
    <Trans id="plugin.manualsprite.card.8.body">Создавай Manual Sprite ассеты прямо из меню правой кнопки → Paper2D.</Trans>
  </Card>
</CardGrid>
