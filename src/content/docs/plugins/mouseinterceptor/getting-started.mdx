## <Trans id="plugin.mouseinterceptor.getting-started.h.установка">Установка</Trans>

<Steps>
  <Step title="Купи и установи через FAB">
    Открой плагин на [FAB Marketplace](https://www.fab.com/listings/) и нажми **Install to Engine** — плагин автоматически скачается в твою версию UE.
  </Step>
  <Step title="Включи плагин в проекте">
    Открой проект в Unreal Editor → **Edit → Plugins → Installed → Input → Mouse Interceptor** → галочка.
  </Step>
  <Step title="Перезапусти редактор">
    Editor предложит перекомпилировать плагин под твой проект и перезапустится автоматически.
  </Step>
</Steps>

---

## <Trans id="plugin.mouseinterceptor.getting-started.h.шаг-1-добавить-компонент">Шаг 1 — Добавить компонент</Trans>

<Steps>
  <Step title="Открой актор">
    Выбери актора, в котором нужен перехват мыши (Character, PlayerController, ActorComponent-хост — любой).
  </Step>
  <Step title="Add Component">
    В панели **Components** нажми **Add Component**, начни вводить `MouseInterceptor`.
  </Step>
  <Step title="Выбери MouseInterceptorComponent">
    Добавь его — компонент появится в иерархии.
  </Step>
</Steps>

<Image src="/images/plugins/mouseinterceptor/Image_01.png" alt="MouseInterceptorComponent в панели Components" caption="Компонент добавлен в актор" />

## <Trans id="plugin.mouseinterceptor.getting-started.h.шаг-2-подписаться-на-делегаты">Шаг 2 — Подписаться на делегаты</Trans>

<Trans id="plugin.mouseinterceptor.getting-started.шаг-2-подписаться-на-делегаты.p1">В **Blueprint Event Graph** выбранного актора:</Trans>

<Steps>
  <Step title="Выбери компонент">
    В Components pane слева щёлкни `MouseInterceptorComponent`.
  </Step>
  <Step title="Найди события">
    В Details panel → секция **Events** → `OnMousePressed` / `OnMouseReleased`. Нажми кнопку **+** справа от нужного — в Event Graph появится узел.
  </Step>
</Steps>

<Image src="/images/plugins/mouseinterceptor/Image_02.png" alt="Привязка OnMousePressed в Event Graph" caption="Узел OnMousePressed создан" />

<Trans id="plugin.mouseinterceptor.getting-started.шаг-2-подписаться-на-делегаты.p2">Параметры делегата:</Trans>

- <Trans id="plugin.mouseinterceptor.getting-started.шаг-2-подписаться-на-делегаты.li1">**`Button`** — `EKeys::LeftMouseButton`, `RightMouseButton`, `MiddleMouseButton` и т.д.</Trans>
- <Trans id="plugin.mouseinterceptor.getting-started.шаг-2-подписаться-на-делегаты.li2">**`bIsDoubleClick`** — `true` если клик попал в окно `DoubleClickThreshold` с предыдущего клика той же кнопкой.</Trans>

<Image src="/images/plugins/mouseinterceptor/Image_03.png" alt="Использование bIsDoubleClick в Blueprint" caption="Ветвление по bIsDoubleClick" />

## <Trans id="plugin.mouseinterceptor.getting-started.h.настройка-порога-двойного-клика">Настройка порога двойного клика</Trans>

<Trans id="plugin.mouseinterceptor.getting-started.настройка-порога-двойного-клика.p1">По умолчанию порог — `0.25` секунды. Меняется в runtime через `SetDoubleClickThreshold`.</Trans>

```cpp
MouseInterceptorComponent->SetDoubleClickThreshold(0.5f); // полсекунды
```

<Trans id="plugin.mouseinterceptor.getting-started.настройка-порога-двойного-клика.p2">Или через Blueprint (тот же метод):</Trans>

<Image src="/images/plugins/mouseinterceptor/Image_05.png" alt="SetDoubleClickThreshold в Blueprint" caption="Метод вызывается как обычная Blueprint-функция" />

<Block type="warning" title="Отключение двойного клика">
<Trans id="plugin.mouseinterceptor.getting-started.block.p1">Плагин всегда определяет двойной клик. Чтобы фактически «отключить» определение — поставь порог очень большим: `SetDoubleClickThreshold(10.0f)`. Тогда флаг `bIsDoubleClick` будет `false` почти всегда.</Trans>
</Block>

## <Trans id="plugin.mouseinterceptor.getting-started.h.заметки">Заметки</Trans>

<Block type="info" title="Client-side only">
<Trans id="plugin.mouseinterceptor.getting-started.block.p1">Плагин обрабатывает ввод на клиенте — сетевая репликация не поддерживается. Если нужен сетевой кликстрим, отправляй события сам через RPC.</Trans>
</Block>

<Block type="ok" title="Zero Tick">
<Trans id="plugin.mouseinterceptor.getting-started.block.p1">Компонент не использует `Tick` — Slate сам вызывает препроцессор только при реальных событиях мыши. Можно вешать на сотни акторов без потерь производительности.</Trans>
</Block>
