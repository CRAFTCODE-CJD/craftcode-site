## <Trans id="plugin.mouseinterceptor.getting-started.h.установка">Установка</Trans>

<Steps>
  <Step title="Купи и установи через FAB" titleId="plugin.mouseinterceptor.getting-started.step.1.title">
    <Trans id="plugin.mouseinterceptor.getting-started.step.1.body">Открой плагин на [FAB Marketplace](https://www.fab.com/listings/8a9a5182-f819-4dfb-9c1f-b7f9a4874dbe) и нажми **Install to Engine** — плагин автоматически скачается в твою версию UE.</Trans>
  </Step>
  <Step title="Включи плагин в проекте" titleId="plugin.mouseinterceptor.getting-started.step.2.title">
    <Trans id="plugin.mouseinterceptor.getting-started.step.2.body">Открой проект в Unreal Editor → **Edit → Plugins → Installed → Input → Mouse Interceptor** → галочка.</Trans>
  </Step>
  <Step title="Перезапусти редактор" titleId="plugin.mouseinterceptor.getting-started.step.3.title">
    <Trans id="plugin.mouseinterceptor.getting-started.step.3.body">Editor предложит перекомпилировать плагин под твой проект и перезапустится автоматически.</Trans>
  </Step>
</Steps>

## <Trans id="plugin.mouseinterceptor.getting-started.h.шаг-1-добавить-компонент">Шаг 1 — Добавить компонент</Trans>

<Steps>
  <Step title="Открой актор" titleId="plugin.mouseinterceptor.getting-started.step.4.title">
    <Trans id="plugin.mouseinterceptor.getting-started.step.4.body">Выбери актора, в котором нужен перехват мыши (Character, PlayerController, ActorComponent-хост — любой).</Trans>
  </Step>
  <Step title="Add Component" titleId="plugin.mouseinterceptor.getting-started.step.5.title">
    <Trans id="plugin.mouseinterceptor.getting-started.step.5.body">В панели **Components** нажми **Add Component**, начни вводить `MouseInterceptor`.</Trans>
  </Step>
  <Step title="Выбери MouseInterceptorComponent" titleId="plugin.mouseinterceptor.getting-started.step.6.title">
    <Trans id="plugin.mouseinterceptor.getting-started.step.6.body">Добавь его — компонент появится в иерархии.</Trans>
  </Step>
</Steps>

## <Trans id="plugin.mouseinterceptor.getting-started.h.шаг-2-подписаться-на-делегаты">Шаг 2 — Подписаться на делегаты</Trans>

<Trans id="plugin.mouseinterceptor.getting-started.шаг-2-подписаться-на-делегаты.p1">В **Blueprint Event Graph** выбранного актора:</Trans>

<Steps>
  <Step title="Выбери компонент" titleId="plugin.mouseinterceptor.getting-started.step.7.title">
    <Trans id="plugin.mouseinterceptor.getting-started.step.7.body">В Components pane слева щёлкни `MouseInterceptorComponent`.</Trans>
  </Step>
  <Step title="Найди события" titleId="plugin.mouseinterceptor.getting-started.step.8.title">
    <Trans id="plugin.mouseinterceptor.getting-started.step.8.body">В Details panel → секция **Events** → `OnMousePressed` / `OnMouseReleased`. Нажми кнопку **+** справа от нужного — в Event Graph появится узел.</Trans>
  </Step>
</Steps>

<Image src="/images/plugins/mouseinterceptor/Image_02.png" alt="Привязка OnMousePressed в Event Graph" caption="Узел OnMousePressed создан" altId="plugin.mouseinterceptor.getting-started.image.2.alt" captionId="plugin.mouseinterceptor.getting-started.image.2.caption" />

<Trans id="plugin.mouseinterceptor.getting-started.шаг-2-подписаться-на-делегаты.p2">Параметры делегата:</Trans>

- <Trans id="plugin.mouseinterceptor.getting-started.шаг-2-подписаться-на-делегаты.li1">**`Button`** — `EKeys::LeftMouseButton`, `RightMouseButton`, `MiddleMouseButton` и т.д.</Trans>
- <Trans id="plugin.mouseinterceptor.getting-started.шаг-2-подписаться-на-делегаты.li2">**`bIsDoubleClick`** — `true` если клик попал в окно `DoubleClickThreshold` с предыдущего клика той же кнопкой.</Trans>

```text
  → Branch: bIsDoubleClick?

```

## <Trans id="plugin.mouseinterceptor.getting-started.h.настройка-порога-двойного-клика">Настройка порога двойного клика</Trans>

<Trans id="plugin.mouseinterceptor.getting-started.настройка-порога-двойного-клика.p1">По умолчанию порог — `0.25` секунды. Меняется в runtime через `SetDoubleClickThreshold`.</Trans>

```cpp
MouseInterceptorComponent->SetDoubleClickThreshold(0.5f); // полсекунды
```

<Trans id="plugin.mouseinterceptor.getting-started.настройка-порога-двойного-клика.p2">Или через Blueprint (тот же метод):</Trans>

<Image src="/images/plugins/mouseinterceptor/Image_05.png" alt="SetDoubleClickThreshold в Blueprint" caption="Метод вызывается как обычная Blueprint-функция" altId="plugin.mouseinterceptor.getting-started.image.5.alt" captionId="plugin.mouseinterceptor.getting-started.image.5.caption" />

<Block type="warning" title="Отключение двойного клика" titleId="plugin.mouseinterceptor.getting-started.block.1.title">
<Trans id="plugin.mouseinterceptor.getting-started.block.p1">Плагин всегда определяет двойной клик. Чтобы фактически «отключить» определение — поставь порог очень большим: `SetDoubleClickThreshold(10.0f)`. Тогда флаг `bIsDoubleClick` будет `false` почти всегда.</Trans>
</Block>

## <Trans id="plugin.mouseinterceptor.getting-started.h.заметки">Заметки</Trans>

<Block type="info" title="Client-side only" titleId="plugin.mouseinterceptor.getting-started.block.2.title">
<Trans id="plugin.mouseinterceptor.getting-started.block.p1">Плагин обрабатывает ввод на клиенте — сетевая репликация не поддерживается. Если нужен сетевой кликстрим, отправляй события сам через RPC.</Trans>
</Block>

<Block type="ok" title="Zero Tick" titleId="plugin.mouseinterceptor.getting-started.block.3.title">
<Trans id="plugin.mouseinterceptor.getting-started.block.p1">Компонент не использует `Tick` — Slate сам вызывает препроцессор только при реальных событиях мыши. Можно вешать на сотни акторов без потерь производительности.</Trans>
</Block>
