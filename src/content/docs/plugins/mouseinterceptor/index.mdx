## Ключевые возможности

<CardGrid>
  <Card title="Глобальный перехват" icon="setting">
    Захватывает все события мыши — независимо от того, что сейчас активно на экране.
  </Card>
  <Card title="Blueprint-делегаты" icon="puzzle">
    `OnMousePressed` и `OnMouseReleased` — подключи в Event Graph за секунду.
  </Card>
  <Card title="Двойной клик" icon="approve-check">
    Настраиваемый порог двойного клика через `SetDoubleClickThreshold`. Отключается высоким значением.
  </Card>
  <Card title="Без тиков" icon="rocket">
    Работает без Tick — минимальная нагрузка на CPU.
  </Card>
</CardGrid>
