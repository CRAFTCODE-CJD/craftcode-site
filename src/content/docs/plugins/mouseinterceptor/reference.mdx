## <Trans id="plugin.mouseinterceptor.reference.h.компонент">Компонент</Trans>

<Trans id="plugin.mouseinterceptor.reference.компонент.p1">**`UMouseInterceptorComponent`** — `UActorComponent`, добавляется к любому актору через панель **Components**.</Trans>

<Block type="info">
<Trans id="plugin.mouseinterceptor.reference.block.p1">Компонент регистрирует собственный `IInputProcessor` в `FSlateApplication` при `BeginPlay` и снимает регистрацию при `EndPlay`. Никаких тиков, никакого polling.</Trans>
</Block>

---

## <Trans id="plugin.mouseinterceptor.reference.h.делегаты">Делегаты</Trans>

### <Trans id="plugin.mouseinterceptor.reference.h.onmousepressed">`OnMousePressed`</Trans>

<Trans id="plugin.mouseinterceptor.reference.onmousepressed.p1">Срабатывает при **нажатии** кнопки мыши.</Trans>

| Параметр | Тип | Описание |
|----------|-----|----------|
| `Button` | `FKey` | Нажатая кнопка (`EKeys::LeftMouseButton`, `RightMouseButton`, `MiddleMouseButton`, `ThumbMouseButton`, `ThumbMouseButton2`) |
| `bIsDoubleClick` | `bool` | `true`, если клик попал в окно `DoubleClickThreshold` с предыдущего клика той же кнопкой |

### <Trans id="plugin.mouseinterceptor.reference.h.onmousereleased">`OnMouseReleased`</Trans>

<Trans id="plugin.mouseinterceptor.reference.onmousereleased.p1">Срабатывает при **отпускании** кнопки мыши.</Trans>

| Параметр | Тип | Описание |
|----------|-----|----------|
| `Button` | `FKey` | Отпущенная кнопка |
| `bIsDoubleClick` | `bool` | Флаг двойного клика для последовательности press+release |

## <Trans id="plugin.mouseinterceptor.reference.h.функции">Функции</Trans>

### <Trans id="plugin.mouseinterceptor.reference.h.setdoubleclickthreshold-float-threshold">`SetDoubleClickThreshold(float Threshold)`</Trans>

<Trans id="plugin.mouseinterceptor.reference.setdoubleclickthreshold-float-threshold.p1">Устанавливает интервал (в секундах) для определения двойного клика.</Trans>

```cpp
MouseInterceptorComponent->SetDoubleClickThreshold(0.5f);
```

<Trans id="plugin.mouseinterceptor.reference.setdoubleclickthreshold-float-threshold.p2">По умолчанию — `0.25s`. Рекомендации:</Trans>

- <Trans id="plugin.mouseinterceptor.reference.setdoubleclickthreshold-float-threshold.li1">`0.2–0.3s` — стандартный «windows-подобный» двойной клик</Trans>
- <Trans id="plugin.mouseinterceptor.reference.setdoubleclickthreshold-float-threshold.li2">`0.15s` — для быстрого геймплея</Trans>
- <Trans id="plugin.mouseinterceptor.reference.setdoubleclickthreshold-float-threshold.li3">`10.0f` — фактическое отключение определения (флаг всегда `false`)</Trans>

### <Trans id="plugin.mouseinterceptor.reference.h.getdoubleclickthreshold">`GetDoubleClickThreshold()`</Trans>

<Trans id="plugin.mouseinterceptor.reference.getdoubleclickthreshold.p1">Возвращает текущий порог в секундах.</Trans>

## <Trans id="plugin.mouseinterceptor.reference.h.faq">FAQ</Trans>

### <Trans id="plugin.mouseinterceptor.reference.h.как-отключить-двойной-клик">Как отключить двойной клик?</Trans>

<Trans id="plugin.mouseinterceptor.reference.как-отключить-двойной-клик.p1">Установи `DoubleClickThreshold` в очень большое значение — например `10.0f`. Флаг `bIsDoubleClick` будет `false` практически всегда.</Trans>

### <Trans id="plugin.mouseinterceptor.reference.h.поддерживается-ли-сетевая-репликация">Поддерживается ли сетевая репликация?</Trans>

<Trans id="plugin.mouseinterceptor.reference.поддерживается-ли-сетевая-репликация.p1">Нет — плагин обрабатывает ввод только на клиентской стороне. Для сетевой передачи кликов оберни делегаты в RPC руками.</Trans>

### <Trans id="plugin.mouseinterceptor.reference.h.работает-ли-перехват-когда-активен-ui-виджет">Работает ли перехват когда активен UI-виджет?</Trans>

<Trans id="plugin.mouseinterceptor.reference.работает-ли-перехват-когда-активен-ui-виджет.p1">Да — перехват **глобальный**, независим от состояния фокуса ввода. Работает когда курсор вне окна? **Нет** — Slate получает события только пока окно активно.</Trans>

### <Trans id="plugin.mouseinterceptor.reference.h.что-с-right-click-ue-меню-контекста">Что с right-click UE-меню контекста?</Trans>

<Trans id="plugin.mouseinterceptor.reference.что-с-right-click-ue-меню-контекста.p1">Плагин получает события **до** стандартной обработки Slate. Если нужно сохранить дефолтное поведение (например, ПКМ в UMG-виджете), обрабатывай в своём делегате флаг `bIsHandled` — можно вернуть `false` и пустить событие дальше.</Trans>

### <Trans id="plugin.mouseinterceptor.reference.h.совместим-с-enhanced-input">Совместим с Enhanced Input?</Trans>

<Trans id="plugin.mouseinterceptor.reference.совместим-с-enhanced-input.p1">Да — плагин работает параллельно Enhanced Input (оба слушают Slate). Два разных слоя ввода, не конкурируют.</Trans>
