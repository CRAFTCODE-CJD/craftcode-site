**Atlas Mode** — основной режим. Пакует несколько текстур в один атлас, генерирует спрайты и Material Instances с пиксельными UV-параметрами.

---

## Открытие редактора

Content Browser → выдели 2+ текстур → ПКМ → **Sprite Optimizer → Atlas**.

![Редактор атласа — Items, Viewport, Settings](/images/plugins/sprite-optimizer/Atlas_01.png)

### Структура редактора

| Панель | Описание |
|--------|----------|
| **Items** (слева) | Список элементов: размещённые (Placed) и не размещённые (Unplaced). Поиск, фильтр, чекбоксы. |
| **Viewport** (центр) | Интерактивный холст — перемещение, поворот, выделение. |
| **Settings** (справа) | Алгоритм, размер страницы, параметры вывода. |

## Тулбар

| Кнопка | Шорткат | Описание |
|--------|---------|----------|
| Analyze | `F5` | Запустить алгоритм пакинга и показать превью |
| Create Atlas | `Ctrl+Shift+Enter` | Экспортировать атлас, спрайты, Material Instances |
| Repack | `Ctrl+R` | Перепаковать текущую страницу |
| Place All | — | Разместить все Unplaced элементы |
| Import Atlas | — | Импортировать готовый атлас |
| Fit | `Home` | Подогнать вид под весь атлас |
| Grid | `G` | Переключить сетку |
| Snap | `S` | Переключить привязку к сетке |
| Free Move | `F` | Свободное перемещение (без коллизий) |

## Алгоритмы пакинга

| Алгоритм | Лучше всего для |
|----------|----------------|
| **Max Rects** | Универсальный. 5 эвристик × 4 порядка сортировки. Отраслевой стандарт. |
| **Bin Pack** | Алгоритм опорных точек. Хорошо для разнородных размеров. |
| **Fixed Grid** | Равномерная N×M сетка. Лучший выбор для tile-based игр. |
| **Tile (Simple)** | Горизонтальная/вертикальная полоса. Для линейных раскладок. |

## Работа с холстом

### Управление

| Действие | Как |
|----------|-----|
| Выбрать элемент | ЛКМ по элементу |
| Добавить в выделение | Ctrl+ЛКМ |
| Rubber band | ЛКМ перетащить по пустому месту |
| Переместить | ЛКМ перетащить по элементу |
| **Axis lock** | Shift+перетаскивание — фиксирует ось (H или V, автодетект) |
| Поворот 90° | `R` на выделенных |
| Убрать из атласа | `Del` — переходит в Unplaced |

### Мультистраничные атласы

Если элементы не помещаются на одной странице, плагин автоматически создаёт дополнительные страницы. Переключение через таб-бар над viewport.

### Коллизии

Пересекающиеся элементы подсвечиваются **красным**, в статус-баре появляется предупреждение "Collisions!". Включи **Free Move** (`F`) чтобы разрешить перекрытия.

### Drag & Drop

Перетащи текстуры из Content Browser прямо в панель Items — они добавятся как Unplaced, затем нажми Place All или Analyze.

## Вывод

После **Create Atlas** (`Ctrl+Shift+Enter`) в Content Browser появляются:

```
Content/YourFolder/
├── MyAtlas.uasset              ← атлас-текстура (BC7 компрессия)
├── MyAtlas_Project.uasset      ← сессия редактирования
├── MyAtlas_Sprite1.uasset      ← PaperSprite с корректными SourceUV
└── MyAtlas_Sprite1_MI.uasset   ← Material Instance с пиксельными UV
```

### Material Instance параметры

| Параметр | Описание |
|----------|----------|
| PixelX, PixelY | Позиция спрайта в атласе (пиксели) |
| SizeW, SizeH | Размер спрайта (пиксели) |
| AtlasW, AtlasH | Размер атласа (пиксели) |
| UVRotation | Флаг поворота (для упакованных с ротацией) |

Работает в 3D-мешах, Paper2D-спрайтах и UMG-виджетах.

## Настройки вывода

| Опция | Описание |
|-------|----------|
| **Texture** | Всегда создаётся — BC7 сжатие с альфа |
| **Sprite** | PaperSprite ассеты |
| **Material Instance** | MI с пиксельными UV параметрами |
| **Preserve Position** | Скорректировать пивот после обрезки |
| **Pad to x4** | Размеры кратны 4 для BC-компрессии |
