<Trans id="plugin.spriteOptimizer.features.atlas.body.p1">**Atlas Mode** — основной режим. Пакует несколько текстур в один атлас, генерирует спрайты и Material Instances с пиксельными UV-параметрами.</Trans>

---

## <Trans id="plugin.spriteOptimizer.features.atlas.h.открытие-редактора">Открытие редактора</Trans>

<Trans id="plugin.spriteOptimizer.features.atlas.открытие-редактора.p1">Content Browser → выдели 2+ текстур → ПКМ → **Sprite Optimizer → Atlas**.</Trans>

![Редактор атласа — Items, Viewport, Settings](/images/plugins/sprite-optimizer/Atlas_01.png)

### <Trans id="plugin.spriteOptimizer.features.atlas.h.структура-редактора">Структура редактора</Trans>

| Панель | Описание |
|--------|----------|
| **Items** (слева) | Список элементов: размещённые (Placed) и не размещённые (Unplaced). Поиск, фильтр, чекбоксы. |
| **Viewport** (центр) | Интерактивный холст — перемещение, поворот, выделение. |
| **Settings** (справа) | Алгоритм, размер страницы, параметры вывода. |

---

## <Trans id="plugin.spriteOptimizer.features.atlas.h.тулбар">Тулбар</Trans>

| Кнопка | Шорткат | Описание |
|--------|---------|----------|
| Analyze | `F5` | Запустить алгоритм пакинга и показать превью |
| Create Atlas | `Ctrl+Shift+Enter` | Экспортировать атлас, спрайты, Material Instances |
| Repack | `Ctrl+R` | Перепаковать текущую страницу |
| Place All | — | Разместить все Unplaced элементы |
| Import Atlas | — | Импортировать готовый атлас |
| Fit | `Home` | Подогнать вид под весь атлас |
| Grid | `G` | Переключить сетку |
| Snap | `S` | Переключить привязку к сетке |
| Free Move | `F` | Свободное перемещение (без коллизий) |

---

## <Trans id="plugin.spriteOptimizer.features.atlas.h.алгоритмы-пакинга">Алгоритмы пакинга</Trans>

| Алгоритм | Лучше всего для |
|----------|----------------|
| **Max Rects** | Универсальный. 5 эвристик × 4 порядка сортировки. Отраслевой стандарт. |
| **Bin Pack** | Алгоритм опорных точек. Хорошо для разнородных размеров. |
| **Fixed Grid** | Равномерная N×M сетка. Лучший выбор для tile-based игр. |
| **Tile (Simple)** | Горизонтальная/вертикальная полоса. Для линейных раскладок. |

## <Trans id="plugin.spriteOptimizer.features.atlas.h.работа-с-холстом">Работа с холстом</Trans>

### <Trans id="plugin.spriteOptimizer.features.atlas.h.управление">Управление</Trans>

| Действие | Как |
|----------|-----|
| Выбрать элемент | ЛКМ по элементу |
| Добавить в выделение | Ctrl+ЛКМ |
| Rubber band | ЛКМ перетащить по пустому месту |
| Переместить | ЛКМ перетащить по элементу |
| **Axis lock** | Shift+перетаскивание — фиксирует ось (H или V, автодетект) |
| Поворот 90° | `R` на выделенных |
| Убрать из атласа | `Del` — переходит в Unplaced |

### <Trans id="plugin.spriteOptimizer.features.atlas.h.мультистраничные-атласы">Мультистраничные атласы</Trans>

<Trans id="plugin.spriteOptimizer.features.atlas.мультистраничные-атласы.p1">Если элементы не помещаются на одной странице, плагин автоматически создаёт дополнительные страницы. Переключение через таб-бар над viewport.</Trans>

### <Trans id="plugin.spriteOptimizer.features.atlas.h.коллизии">Коллизии</Trans>

<Trans id="plugin.spriteOptimizer.features.atlas.коллизии.p1">Пересекающиеся элементы подсвечиваются **красным**, в статус-баре появляется предупреждение "Collisions!". Включи **Free Move** (`F`) чтобы разрешить перекрытия.</Trans>

### <Trans id="plugin.spriteOptimizer.features.atlas.h.drag-drop">Drag & Drop</Trans>

<Trans id="plugin.spriteOptimizer.features.atlas.drag-drop.p1">Перетащи текстуры из Content Browser прямо в панель Items — они добавятся как Unplaced, затем нажми Place All или Analyze.</Trans>

## <Trans id="plugin.spriteOptimizer.features.atlas.h.вывод">Вывод</Trans>

<Trans id="plugin.spriteOptimizer.features.atlas.вывод.p1">После **Create Atlas** (`Ctrl+Shift+Enter`) в Content Browser появляются:</Trans>

```
Content/YourFolder/
├── MyAtlas.uasset              ← атлас-текстура (BC7 компрессия)
├── MyAtlas_Project.uasset      ← сессия редактирования
├── MyAtlas_Sprite1.uasset      ← PaperSprite с корректными SourceUV
└── MyAtlas_Sprite1_MI.uasset   ← Material Instance с пиксельными UV
```

### <Trans id="plugin.spriteOptimizer.features.atlas.h.material-instance-параметры">Material Instance параметры</Trans>

| Параметр | Описание |
|----------|----------|
| PixelX, PixelY | Позиция спрайта в атласе (пиксели) |
| SizeW, SizeH | Размер спрайта (пиксели) |
| AtlasW, AtlasH | Размер атласа (пиксели) |
| UVRotation | Флаг поворота (для упакованных с ротацией) |

<Trans id="plugin.spriteOptimizer.features.atlas.material-instance-параметры.p2">Работает в 3D-мешах, Paper2D-спрайтах и UMG-виджетах.</Trans>

## <Trans id="plugin.spriteOptimizer.features.atlas.h.настройки-вывода">Настройки вывода</Trans>

| Опция | Описание |
|-------|----------|
| **Texture** | Всегда создаётся — BC7 сжатие с альфа |
| **Sprite** | PaperSprite ассеты |
| **Material Instance** | MI с пиксельными UV параметрами |
| **Preserve Position** | Скорректировать пивот после обрезки |
| **Pad to x4** | Размеры кратны 4 для BC-компрессии |
