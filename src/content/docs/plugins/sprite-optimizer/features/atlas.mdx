### <Trans id="plugin.spriteOptimizer.features.atlas.h.мультистраничные-атласы">Мультистраничные атласы</Trans>

<Trans id="plugin.spriteOptimizer.features.atlas.мультистраничные-атласы.p1">Если элементы не помещаются на одной странице, плагин автоматически создаёт дополнительные страницы. Переключение через таб-бар над viewport.</Trans>

### <Trans id="plugin.spriteOptimizer.features.atlas.h.коллизии">Коллизии</Trans>

<Trans id="plugin.spriteOptimizer.features.atlas.коллизии.p1">Пересекающиеся элементы подсвечиваются **красным**, в статус-баре появляется предупреждение "Collisions!". Включи **Free Move** (`F`) чтобы разрешить перекрытия.</Trans>

### <Trans id="plugin.spriteOptimizer.features.atlas.h.drag-drop">Drag & Drop</Trans>

<Trans id="plugin.spriteOptimizer.features.atlas.drag-drop.p1">Перетащи текстуры из Content Browser прямо в панель Items — они добавятся как Unplaced, затем нажми Place All или Analyze.</Trans>

## <Trans id="plugin.spriteOptimizer.features.atlas.h.вывод">Вывод</Trans>

<Trans id="plugin.spriteOptimizer.features.atlas.вывод.p1">После **Create Atlas** (`Ctrl+Shift+Enter`) в Content Browser появляются:</Trans>

```
Content/YourFolder/
├── MyAtlas.uasset              ← атлас-текстура (BC7 компрессия)
├── MyAtlas_Project.uasset      ← сессия редактирования
├── MyAtlas_Sprite1.uasset      ← PaperSprite с корректными SourceUV
└── MyAtlas_Sprite1_MI.uasset   ← Material Instance с пиксельными UV
```
