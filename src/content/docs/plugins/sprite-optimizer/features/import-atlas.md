**Import Atlas** разбивает существующую атлас-текстуру обратно в отдельные элементы, которые можно снова редактировать в Atlas Editor.

---

## Как открыть

Content Browser → ПКМ на атлас-текстуре → **Sprite Optimizer → Import Atlas**.

---

## Источники данных

Если атлас уже имеет привязанные Sprites или Material Instances, появится диалог выбора источника:

- **Yes (Sprites)** — восстановить регионы из данных спрайтов
- **No (Material Instances)** — восстановить из MI UV-параметров
- **Cancel** — открыть Slice Texture вручную

Если метаданных нет — автоматически открывается диалог **Slice Texture**.

---

## Slice Texture — ручная нарезка

### Auto Mode (Flood Fill)

Автоматически определяет регионы по прозрачности (flood fill). Жёлтые контуры = обнаруженные острова.

Настраивается:
- **Alpha Threshold** — порог прозрачности

### Grid Mode

Нарезает текстуру равномерной сеткой.

Настраивается:
- **Cell Width / Cell Height** — размер ячейки
- **Margin** — внешний отступ
- **Spacing** — промежуток между ячейками

### Ручное редактирование регионов

| Действие | Как |
|----------|-----|
| Выбрать регион | ЛКМ |
| Нарисовать новый | ЛКМ перетащить по пустому месту |
| Удалить | Выбрать → `Del` |
| Изменить размер | Потащить края/углы выделенного |
| Отметить как повёрнутый | Выбрать → `R` (снимается при экспорте) |
| Зум / Пан | Колесо мыши / ПКМ или СКМ |

Зелёный = выбранный регион. Оранжевый с "R" = помечен как повёрнутый.

---

## После нарезки

Нажми **Slice** — текстуры сохраняются в подпапку `Sliced/`, и открывается новый Atlas Editor с этими элементами.
