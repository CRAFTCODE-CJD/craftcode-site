---

<Trans id="plugin.spriteOptimizer.features.import-atlas.body.p1">**Import Atlas** разбивает существующую атлас-текстуру обратно в отдельные элементы, которые можно снова редактировать в Atlas Editor.</Trans>

---

## <Trans id="plugin.spriteOptimizer.features.import-atlas.h.как-открыть">Как открыть</Trans>

<Trans id="plugin.spriteOptimizer.features.import-atlas.как-открыть.p1">Content Browser → ПКМ на атлас-текстуре → **Sprite Optimizer → Import Atlas**.</Trans>

---

## <Trans id="plugin.spriteOptimizer.features.import-atlas.h.источники-данных">Источники данных</Trans>

<Trans id="plugin.spriteOptimizer.features.import-atlas.источники-данных.p1">Если атлас уже имеет привязанные Sprites или Material Instances, появится диалог выбора источника:</Trans>

- <Trans id="plugin.spriteOptimizer.features.import-atlas.источники-данных.li1">**Yes (Sprites)** — восстановить регионы из данных спрайтов</Trans>
- <Trans id="plugin.spriteOptimizer.features.import-atlas.источники-данных.li2">**No (Material Instances)** — восстановить из MI UV-параметров</Trans>
- <Trans id="plugin.spriteOptimizer.features.import-atlas.источники-данных.li3">**Cancel** — открыть Slice Texture вручную</Trans>

<Trans id="plugin.spriteOptimizer.features.import-atlas.источники-данных.p2">Если метаданных нет — автоматически открывается диалог **Slice Texture**.</Trans>

---

## <Trans id="plugin.spriteOptimizer.features.import-atlas.h.slice-texture-ручная-нарезка">Slice Texture — ручная нарезка</Trans>

### <Trans id="plugin.spriteOptimizer.features.import-atlas.h.auto-mode-flood-fill">Auto Mode (Flood Fill)</Trans>

<Trans id="plugin.spriteOptimizer.features.import-atlas.auto-mode-flood-fill.p1">Автоматически определяет регионы по прозрачности (flood fill). Жёлтые контуры = обнаруженные острова.</Trans>

<Trans id="plugin.spriteOptimizer.features.import-atlas.auto-mode-flood-fill.p2">Настраивается:</Trans>
- <Trans id="plugin.spriteOptimizer.features.import-atlas.auto-mode-flood-fill.li1">**Alpha Threshold** — порог прозрачности</Trans>

### <Trans id="plugin.spriteOptimizer.features.import-atlas.h.grid-mode">Grid Mode</Trans>

<Trans id="plugin.spriteOptimizer.features.import-atlas.grid-mode.p1">Нарезает текстуру равномерной сеткой.</Trans>

<Trans id="plugin.spriteOptimizer.features.import-atlas.grid-mode.p2">Настраивается:</Trans>
- <Trans id="plugin.spriteOptimizer.features.import-atlas.grid-mode.li1">**Cell Width / Cell Height** — размер ячейки</Trans>
- <Trans id="plugin.spriteOptimizer.features.import-atlas.grid-mode.li2">**Margin** — внешний отступ</Trans>
- <Trans id="plugin.spriteOptimizer.features.import-atlas.grid-mode.li3">**Spacing** — промежуток между ячейками</Trans>

## <Trans id="plugin.spriteOptimizer.features.import-atlas.h.после-нарезки">После нарезки</Trans>

<Trans id="plugin.spriteOptimizer.features.import-atlas.после-нарезки.p1">Нажми **Slice** — текстуры сохраняются в подпапку `Sliced/`, и открывается новый Atlas Editor с этими элементами.</Trans>
