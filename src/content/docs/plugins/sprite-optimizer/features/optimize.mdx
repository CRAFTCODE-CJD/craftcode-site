<Trans id="plugin.spriteOptimizer.features.optimize.body.p1">**Optimize Mode** обрезает прозрачные пиксели по краям текстур, значительно уменьшая их размер — при этом спрайты остаются на тех же визуальных позициях в сцене.</Trans>

---

## <Trans id="plugin.spriteOptimizer.features.optimize.h.как-открыть">Как открыть</Trans>

<Trans id="plugin.spriteOptimizer.features.optimize.как-открыть.p1">Content Browser → выдели текстуры → ПКМ → **Sprite Optimizer → Optimize**.</Trans>

---

## <Trans id="plugin.spriteOptimizer.features.optimize.h.рабочий-процесс">Рабочий процесс</Trans>

1. Редактор откроется со списком выбранных текстур.
2. Нажми **F5** (Analyze) — плагин вычислит обрезанные размеры и экономию памяти.
3. Просмотри результаты: **жёлтый** = хорошая экономия, **оранжевый** = размер вырос (padding увеличил площадь).
4. При желании скорректируй **Alpha Threshold** — граничное значение прозрачности.
5. Нажми **Ctrl+Shift+Enter** — обрезанные текстуры сохранятся.

## <Trans id="plugin.spriteOptimizer.features.optimize.h.сохранение-пивота-preserve-position">Сохранение пивота (Preserve Position)</Trans>

<Trans id="plugin.spriteOptimizer.features.optimize.сохранение-пивота-preserve-position.p1">Опция **Preserve Pos ✓** — ключевая фича. После обрезки плагин автоматически корректирует пивот спрайта так, чтобы он оставался на прежней визуальной позиции в сцене.</Trans>

![Демонстрация Preserve Position — 99% экономия памяти](/images/plugins/sprite-optimizer/PivotPreservation_Demo.png)

<Trans id="plugin.spriteOptimizer.features.optimize.сохранение-пивота-preserve-position.p2">Реальный пример из README:</Trans>

> 6 текстур 2048×2048 обрезаны до фактического содержимого.  
> **32.2 MB → 186 KB** (экономия 99%).  
> Сцена не сломалась — всё стоит на своих местах.

## <Trans id="plugin.spriteOptimizer.features.optimize.h.настройки">Настройки</Trans>

| Параметр | Описание |
|----------|----------|
| **Alpha Threshold** | Пиксели с альфой ≤ порога считаются прозрачными и обрезаются |
| **Preserve Position** | Скорректировать пивот чтобы спрайт не сдвинулся визуально |
| **Pad to x4** | Добавить отступ до кратности 4 — нужно для BC-компрессии |
| **Output subfolder** | Куда сохранять обрезанные текстуры (настраивается в Project Settings) |

---

## <Trans id="plugin.spriteOptimizer.features.optimize.h.советы">Советы</Trans>

- **Alpha Threshold по умолчанию = 0** (полностью прозрачные пиксели). Увеличь до 1–5 чтобы срезать полупрозрачную "ауру" вокруг объектов.
- Если после обрезки размер *вырос* (оранжевый цвет) — текстура практически без прозрачных краёв, обрезать её нет смысла.
- Обрезка работает на оригинале **без изменений** — исходный файл не трогается, создаётся новый.
