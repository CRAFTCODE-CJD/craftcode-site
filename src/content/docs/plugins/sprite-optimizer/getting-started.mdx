## <Trans id="plugin.spriteOptimizer.getting-started.h.установка">Установка</Trans>

<Trans id="plugin.spriteOptimizer.getting-started.установка.p1">Плагин ставится через **FAB Marketplace** — после покупки он доступен через Epic Games Launcher.</Trans>

1. <Trans id="plugin.spriteOptimizer.getting-started.установка.li1">Открой **Epic Games Launcher → Библиотека → Vault**.</Trans>
2. <Trans id="plugin.spriteOptimizer.getting-started.установка.li2">Найди **Sprite Optimizer** и нажми **Install to Engine** (или **Add to project**).</Trans>
3. <Trans id="plugin.spriteOptimizer.getting-started.установка.li3">Перезапусти редактор.</Trans>
4. <Trans id="plugin.spriteOptimizer.getting-started.установка.li4">В меню **Edit → Plugins** убедись, что **Sprite Optimizer** включён.</Trans>

> Поддерживаются UE 4.27 и 5.0–5.7. Один плагин — все версии.

---

## <Trans id="plugin.spriteOptimizer.getting-started.h.первые-шаги">Первые шаги</Trans>

### <Trans id="plugin.spriteOptimizer.getting-started.h.шаг-1-выбери-текстуры">Шаг 1 — выбери текстуры</Trans>

<Trans id="plugin.spriteOptimizer.getting-started.шаг-1-выбери-текстуры.p1">Открой **Content Browser**, выдели одну или несколько текстур/спрайтов.</Trans>

### <Trans id="plugin.spriteOptimizer.getting-started.h.шаг-2-контекстное-меню">Шаг 2 — контекстное меню</Trans>

<Trans id="plugin.spriteOptimizer.getting-started.шаг-2-контекстное-меню.p1">Нажми **ПКМ** → в секции **Sprite Optimizer** выбери нужное действие:</Trans>

| Действие | Описание |
|----------|----------|
| **Optimize** | Обрезать прозрачные края |
| **Atlas** | Открыть полный редактор атласа |
| **Quick Atlas** | Создать атлас сразу на настройках по умолчанию |
| **Import Atlas** | Реверс-инжиниринг готового атласа |

<Trans id="plugin.spriteOptimizer.getting-started.шаг-2-контекстное-меню.p3">При выборе нескольких текстур доступны Optimize, Atlas, Quick Atlas, Analyze.  
При выборе одной — дополнительно Import Atlas.</Trans>

## <Trans id="plugin.spriteOptimizer.getting-started.h.создание-первого-атласа">Создание первого атласа</Trans>

1. <Trans id="plugin.spriteOptimizer.getting-started.создание-первого-атласа.li1">Выдели 2+ текстуры в Content Browser.</Trans>
2. <Trans id="plugin.spriteOptimizer.getting-started.создание-первого-атласа.li2">ПКМ → **Atlas** — откроется редактор.</Trans>
3. <Trans id="plugin.spriteOptimizer.getting-started.создание-первого-атласа.li3">Нажми **F5** (Analyze) — алгоритм расставит текстуры по атласу.</Trans>
4. <Trans id="plugin.spriteOptimizer.getting-started.создание-первого-атласа.li4">При желании перетащи элементы вручную, измени размер страницы.</Trans>
5. <Trans id="plugin.spriteOptimizer.getting-started.создание-первого-атласа.li5">Нажми **Ctrl+Shift+Enter** (Create Atlas) — атлас, спрайты и Material Instances сохранятся рядом с исходниками.</Trans>

```
Content/YourFolder/
├── MyAtlas.uasset              ← атлас-текстура
├── MyAtlas_Project.uasset      ← сессия редактирования (открыть двойным кликом)
├── MyAtlas_Sprite1.uasset      ← сгенерированный спрайт
└── MyAtlas_Sprite1_MI.uasset   ← Material Instance с UV-параметрами
```

## <Trans id="plugin.spriteOptimizer.getting-started.h.быстрый-атлас-quick-atlas">Быстрый атлас (Quick Atlas)</Trans>

<Trans id="plugin.spriteOptimizer.getting-started.быстрый-атлас-quick-atlas.p1">Для простых случаев — выдели 2+ текстур, ПКМ → **Quick Atlas**. Атлас создаётся мгновенно на настройках из Project Settings без открытия редактора.</Trans>
