## Полный список функций

- **Автоматическая обрезка** — убирает прозрачные края, настраиваемый alpha threshold
- **Сохранение пивота** — обрезанные спрайты визуально остаются на месте в сцене
- **4 алгоритма пакинга** — Max Rects, Bin Pack, Fixed Grid, Tile
- **Import Atlas** — реверс-инжиниринг готового атласа обратно в отдельные элементы
- **Quick Atlas** — атлас в один клик на настройках по умолчанию
- **Material Instances** — автогенерация с pixel-based UV параметрами, работает в 3D и UMG
- **Мультистраничные атласы** — автосоздание дополнительных страниц при нехватке места
- **Normal maps & HDR** — 16-битные источники и HDR текстуры пакуются корректно
- **BC7 компрессия** — высококачественное сжатие с поддержкой альфа-канала
- **Memory warnings** — предупреждение с оценкой RAM перед тяжёлыми операциями
- **50 уровней undo/redo** — экспериментируй без страха
- **Нулевой runtime-cost** — плагин editor-only, в пакованную игру не попадает
