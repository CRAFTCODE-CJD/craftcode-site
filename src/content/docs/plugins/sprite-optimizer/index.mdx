## <Trans id="plugin.spriteOptimizer.h.полный-список-функций">Полный список функций</Trans>

- <Trans id="plugin.spriteOptimizer.полный-список-функций.li1">**Автоматическая обрезка** — убирает прозрачные края, настраиваемый alpha threshold</Trans>
- <Trans id="plugin.spriteOptimizer.полный-список-функций.li2">**Сохранение пивота** — обрезанные спрайты визуально остаются на месте в сцене</Trans>
- <Trans id="plugin.spriteOptimizer.полный-список-функций.li3">**4 алгоритма пакинга** — Max Rects, Bin Pack, Fixed Grid, Tile</Trans>
- <Trans id="plugin.spriteOptimizer.полный-список-функций.li4">**Import Atlas** — реверс-инжиниринг готового атласа обратно в отдельные элементы</Trans>
- <Trans id="plugin.spriteOptimizer.полный-список-функций.li5">**Quick Atlas** — атлас в один клик на настройках по умолчанию</Trans>
- <Trans id="plugin.spriteOptimizer.полный-список-функций.li6">**Material Instances** — автогенерация с pixel-based UV параметрами, работает в 3D и UMG</Trans>
- <Trans id="plugin.spriteOptimizer.полный-список-функций.li7">**Мультистраничные атласы** — автосоздание дополнительных страниц при нехватке места</Trans>
- <Trans id="plugin.spriteOptimizer.полный-список-функций.li8">**Normal maps & HDR** — 16-битные источники и HDR текстуры пакуются корректно</Trans>
- <Trans id="plugin.spriteOptimizer.полный-список-функций.li9">**BC7 компрессия** — высококачественное сжатие с поддержкой альфа-канала</Trans>
- <Trans id="plugin.spriteOptimizer.полный-список-функций.li10">**Memory warnings** — предупреждение с оценкой RAM перед тяжёлыми операциями</Trans>
- <Trans id="plugin.spriteOptimizer.полный-список-функций.li11">**50 уровней undo/redo** — экспериментируй без страха</Trans>
- <Trans id="plugin.spriteOptimizer.полный-список-функций.li12">**Нулевой runtime-cost** — плагин editor-only, в пакованную игру не попадает</Trans>
