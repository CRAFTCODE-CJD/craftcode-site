### <Trans id="plugin.spriteOptimizer.reference.h.глобальные">Глобальные</Trans>

| Шорткат | Действие |
|---------|----------|
| `F5` | Analyze (запустить пакинг / оптимизацию) |
| `Ctrl+Shift+Enter` | Create Atlas / Execute Optimize |
| `Ctrl+R` | Repack текущей страницы |
| `Ctrl+Z` | Undo |
| `Ctrl+Y` | Redo |
| `Ctrl+A` | Выбрать все элементы на текущей странице |

### <Trans id="plugin.spriteOptimizer.reference.h.навигация-по-холсту">Навигация по холсту</Trans>

| `СКМ перетащить` | Пан |
| `ПКМ перетащить` | Пан (альтернатива) |

## <Trans id="plugin.spriteOptimizer.reference.h.project-settings">Project Settings</Trans>

<Trans id="plugin.spriteOptimizer.reference.project-settings.p1">**Edit → Project Settings → Plugins → Sprite Optimizer**</Trans>

| Категория | Настройки |
|-----------|-----------|
| **Optimization** | Pixels Per Unit по умолчанию, Padding, Alpha Threshold |
| **Auto Selection** | Min Savings Threshold для авто-выбора |
| **Quick Atlas** | Алгоритм, Max Size, Size Mode, Padding, Optimize First, Allow Rotation, Auto Multi-Pack, Pad to x4, Preserve Position, Suffix |
| **Material** | Sprite Material по умолчанию, Texture Parameter Name, Custom Parent Materials, Material Domain |
| **Output** | Имена подпапок, Suffix для оптимизированных файлов |
| **Workflow** | Показ детальных логов, уведомлений, Auto Refresh Content Browser |

### <Trans id="plugin.spriteOptimizer.reference.h.preview-quality">Preview Quality</Trans>

<Trans id="plugin.spriteOptimizer.reference.preview-quality.p1">**Edit → Project Settings → Plugins → Sprite Optimizer → Editor Performance → Preview Quality**</Trans>

| Настройка | Применение |
|-----------|-----------|
| **Low** (512 px) | Много 4K+ источников, ограниченная RAM |
| **Medium** (1024 px) | По умолчанию. Рекомендуется для большинства проектов |
| **High** (2048 px) | Чёткие превью при сильном зуме |
| **Full** | Без ограничений — полное разрешение источника |

> Влияет только на превью в редакторе. Финальный атлас всегда экспортируется в полном разрешении.

## <Trans id="plugin.spriteOptimizer.reference.h.технические-детали">Технические детали</Trans>

| Параметр | Значение |
|----------|----------|
| Поддерживаемые движки | UE 4.27, 5.0–5.7 |
| Тип | Editor-only плагин |
| Зависимости | Paper2D (стандартный плагин движка) |
| Платформы | Windows, Mac, Linux |
| Max размер атласа | До 8192×8192 (настраивается) |
| Компрессия | BC7 с альфа для всех экспортируемых текстур |
| Runtime Cost | **Нуль** — не попадает в пакованную игру |
| Исходные файлы | **Никогда не изменяются** — оригиналы всегда сохраняются |
| Undo/Redo | 50 уровней |

## <Trans id="plugin.spriteOptimizer.reference.h.troubleshooting">Troubleshooting</Trans>

### <Trans id="plugin.spriteOptimizer.reference.h.normal-maps-hdr-текстуры-выглядят-пустыми">Normal maps / HDR-текстуры выглядят пустыми</Trans>

<Trans id="plugin.spriteOptimizer.reference.normal-maps-hdr-текстуры-выглядят-пустыми.p1">Исправлено в **v2.1**. Старые версии поддерживали только 8-битные источники. Обновись до v2.1.</Trans>

### <Trans id="plugin.spriteOptimizer.reference.h.проект-медленно-открывается-на-больших-атласах">Проект медленно открывается на больших атласах</Trans>

<Trans id="plugin.spriteOptimizer.reference.проект-медленно-открывается-на-больших-атласах.p1">Исправлено в **v2.1** (параллельная загрузка). Если всё ещё медленно — снизь **Preview Quality** до Low.</Trans>

### <Trans id="plugin.spriteOptimizer.reference.h.редактор-заканчивает-память">Редактор заканчивает память</Trans>

<Trans id="plugin.spriteOptimizer.reference.редактор-заканчивает-память.p1">Установи **Preview Quality → Low** (512 px cap). На финальном качестве атласа не сказывается.</Trans>

### <Trans id="plugin.spriteOptimizer.reference.h.старые-ошибки-материалов-на-ue-4-27-partially-lo">Старые ошибки материалов на UE 4.27 ("partially loaded")</Trans>

<Trans id="plugin.spriteOptimizer.reference.старые-ошибки-материалов-на-ue-4-27-partially-lo.p1">Исправлено в **v2.1**. Плагин обнаруживает устаревшие кешированные материалы и автоматически их пересоздаёт.</Trans>

### <Trans id="plugin.spriteOptimizer.reference.h.slice-texture-тормозит-при-зуме">Slice Texture тормозит при зуме</Trans>

<Trans id="plugin.spriteOptimizer.reference.slice-texture-тормозит-при-зуме.p1">Исправлено в **v2.1**. Фон-шахматка теперь рендерится только в видимой области.</Trans>

### <Trans id="plugin.spriteOptimizer.reference.h.не-хватает-тулбара-на-ue-4-27">Не хватает тулбара на UE 4.27</Trans>

<Trans id="plugin.spriteOptimizer.reference.не-хватает-тулбара-на-ue-4-27.p1">Исправлено в **v2.1**. Тулбар включён в дефолтный лейаут.</Trans>

### <Trans id="plugin.spriteOptimizer.reference.h.v2-1-cross-engine-release-апрель-2026">v2.1 — Cross-Engine Release (апрель 2026)</Trans>

- **Sprite Optimizer Project** — возобновляемые сессии

- Normal maps & HDR — корректный пакинг 16-битных текстур
- Параллельная загрузка проектов — быстрее на больших атласах
- Исправление краша при регенерации материалов, зависания Slice Texture, ошибок "partially loaded"

- Интерактивный Atlas Editor с визуальным холстом
- Multi-pack, Slice Texture, Material Instances
